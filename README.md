# picture-backend

#### 介绍
相册项目的后端部分

#### 使用说明

1. 创建picture数据库
2. 运行resources/sql文件夹下的sql文件

 **开发时请新建一个分支** 

#### 结构说明

- annotation 注解
- config 配置文件
- controller 控制器
- dao mapper接口
- entity 实体类
    - vo 返回前端的实体类
- enums 枚举
- exception 异常
- interceptor 拦截器
- service service接口
    - impl service实现类
- utils 工具包

- resources 
    - mapper mapper映射文件(xml)
    - sql 初始化sql
    - application.yml 配置文件
